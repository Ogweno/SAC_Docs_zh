## velocityroset

### 概要

控制剖面图中速度roset的放置

### 语法

``` {.bash}
VELOCITYROSET [ON|OFF] [LOCATION UL|UR|LL|LR]
```
``` {.bash}
VR [ON|OFF] [L UL|UR|LL|LR]
```

### 输入

- `ON|OFF`: 打开/关闭速度roset绘制选项
- `LOCATION UL|UR|LL|LR`: 修改速度roset的放置位置。分别对应左上、右上、左下、右下角。

### 缺省值

``` {.bash}
velocityroset off location ll
```
